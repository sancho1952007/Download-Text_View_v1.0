Errors - Text View

Note

(This Error Handling Feature is only Available in versions Text View v2.1 and after...)

Reason: The Error Might Be Due to a File Handling Error.
Solution: Try Using Another File.


Reason: Maybe You Are Trying to Open A Binary File (.exe, .png, etc...).
Solution: Don't Open A Binary File.


Reason: Update Checker Error.
Solution: Re-install Text View.


Reason: Microsoft Smartscreen/Microsoft Defender Error.
Solution: Allow Microsoft To Run This App, This App is Completely Safe (If Downloaded From the Official Website).



Note: Current Text View Versions Does Not Support Binary Reading/Writing.
The Text View Window Will Hang When Your Browser is open for a Text View Update.
