Text View v2.0

About

Text View is Built By: Sancho Godinho.
Text View is a Text Editor Built in Python/Tkinter.
It Has special features like:

1) Auto Coding Language Detect.
2) Change Text Colour.
3) Change Background colour.
4) Check For Updates.
5) Transparent Effect Added.
6) New Windows toolbox window Added.
7) New GUI Design Than v1.0.
8) New "Dark Mode" And "Auto Dark Mode Detect" Added.
9) New Settings GUI Design Added.
10) Error Handling Added.
